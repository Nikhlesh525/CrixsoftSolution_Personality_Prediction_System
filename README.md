# Personality Prediction System Through CV Analysis

A Machine Learning and NLP-based system that analyzes resumes/CVs and predicts personality traits such as Leadership, Communication, Teamwork, Creativity, and Problem-Solving skills.

## Project Overview

This project uses keyword-based Natural Language Processing (NLP) techniques to analyze CV/Resume text and identify personality traits. The system evaluates different aspects of a candidate's profile and generates a personality analysis report.

## Features

- CV Text Analysis
- Keyword-Based Personality Prediction
- Leadership Detection
- Teamwork Analysis
- Communication Skill Assessment
- Creativity Evaluation
- Problem-Solving Assessment
- Personality Analysis Report Generation

## Technologies Used

- Python
- Natural Language Processing (NLP)
- Machine Learning Concepts

## How to Run

1. Open the project in Python/Pydroid 3.
2. Run the `main.py` file.
3. Paste the CV/Resume text when prompted.
4. View the generated personality analysis report.

## Sample Input

```
I am a creative and motivated Artificial Intelligence student with strong leadership and communication skills. I have worked in team projects and enjoy problem solving and research activities.
```

## Sample Output

```
===== Personality Analysis Report =====

Leadership: 2
Teamwork: 1
Communication: 1
Creativity: 1
Problem Solving: 2

Dominant Personality Trait: Leadership
```
